Detailed Description

A dialog that is used to display output from a child process that runs for a short time.

Useful for spawning compilers and other utility programs. Supports input. This is disabled by default. Output piped through stderror is displayed as red, whereas regular output is displayed as black. Note: you must use ShowModal.
