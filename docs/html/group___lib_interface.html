DLLEXPORT IBaseInterface* CreateInterface ( const char *const  pszName,
IFaceResult *  pResult = nullptr 
)

Creates instances of interfaces. This function is implemented in each library.

Parameters
pszNameName of the interface to instantiate.
pResultIf non-null, this will be set to the result of the creation. In the event that an interface instantiation returns null, this can be used to detect whether it found the interface or not.
Returns
Pointer to the interface, or null if the interface couldn't be found or created.
